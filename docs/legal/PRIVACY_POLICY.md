## Data Subject Rights (GDPR)

If you process personal data, data subjects have rights:

### Right of Access (Article 15)

Data subjects can request:
- Confirmation of data processing
- Access to their personal data
- Information about processing purposes

### Right to Rectification (Article 16)

Data subjects can request:
- Correction of inaccurate data
- Completion of incomplete data

### Right to Erasure (Article 17)

Data subjects can request:
- Deletion of their personal data
- "Right to be forgotten"

### Right to Restriction (Article 18)

Data subjects can request:
- Restriction of processing

### Right to Data Portability (Article 20)

Data subjects can request:
- Data in machine-readable format
- Transfer to another controller

### Right to Object (Article 21)

Data subjects can object to:
- Processing for specific purposes

### Automated Decision-Making (Article 22)

Data subjects have rights regarding:
- Automated decisions without human intervention

**You are responsible** for implementing processes to honor these rights.

## Data Breach Notification

If you experience a data breach involving personal data:

### Your Obligations (GDPR)

- **72 hours**: Notify supervisory authority (if high risk)
- **Without undue delay**: Notify affected data subjects (if high risk)
- **Document**: All data breaches, regardless of notification requirement

### Breach Response

1. **Contain**: Stop the breach immediately
2. **Assess**: Determine scope and impact
3. **Notify**: Authorities and affected individuals (if required)
4. **Remediate**: Fix vulnerability and prevent recurrence
5. **Document**: Maintain breach records
